# How to Develop

## Requirements

- Ruby

## Procedures

1. Install Jekyll: `gem install bundler jekyll`
2. Move to the folder: `cd portfolio-site`
3. Install gems: `bundle install`
4. Update Jekyll: `bundle update jekyll`
5. Start the Jekyll server: `bundle exec jekyll serve`
